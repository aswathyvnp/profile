The Basic Language of the Web: HTML

Posted by Laura Jones on Monday June 21st 2027

All modern websites and web applications are bulit using three fundamental technologies:HTML, CSS, JavaScript.These are the languages of the web.
In this post, let's focus on HTML.We will learn what HTML is about and why you too should learn it.

What is HTML?

HTML stands for HyperText Markup Language. Its a markup language that web developers use to structure and describe the webpage(not a programming language).
HTML consist of elements that describe the different type of content: paragragh, link, heading,images,video,etc.Web browsers understand HTML and render HTML code as websites
In HTML each element is made up of 3 part:

  1. The opening tag
  2. The closing tag
  3. The actual element

You can learn more at MDN Web Docs.

Why should you learn HTML?

There are countless reason for learning the fundamnetal language of the web. Here are 5 of them.:

  • To be able to use the fundamental web dev language
  • To hand-craft beautiful websites instead of relying on tools like Wordpress or Wix
  • To bulit web applications
  • To impress friends
